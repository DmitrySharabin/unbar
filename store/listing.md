## Description

One click. No address bar.

Unbar moves your current tab into a minimal popup window — no address bar, no tab strip, no bookmarks bar. Just your content.

HOW TO USE
1. Navigate to any page.
2. Click the Unbar icon in the toolbar.
3. Pick a size — the tab moves into a clean window.

That's it. No configuration required.

FEATURES
• Size presets on number keys 1–9, fully editable in place. Arrow keys and full keyboard navigation.
• Aspect-ratio lock that recognizes common ratios (16:9, 16:10, 4:3, 21:9, and more).
• Keyboard shortcut for instant unbar at your last-used size.
• Presets saved across sessions.

PERFECT FOR
• Presentations — share a clean window without browser clutter.
• Live demos — keep the audience focused on your app, not the URL bar.
• Screen recording — professional captures without distracting UI.
• Focused browsing — fewer distractions while reading or working.

HOW IT WORKS
Unbar relocates your tab into a popup-type window using the Chrome Extensions API. The page doesn't reload, so your scroll position, form input, and JavaScript state are preserved.

PRIVACY
Unbar collects zero data. No analytics, no tracking, no network requests. Its only permissions are activeTab (scoped to the tab you act on) and storage (to save your presets locally).
